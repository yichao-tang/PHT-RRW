# PHT-RRW
This code runs a two-stage robust reversible watermarking using polar harmonic transform for high robustness and capacity.

For more details, see the following paper. C. Wang, Y. Tang, K. Li, S. Bian and Q. Huang, "A two-stage robust reversible watermarking using polar harmonic transform for high robustness and capacity"
